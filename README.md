Inspired by an algo class. Written in TypeScript, uses the bun runtime, Pug HTML template engine, and tailwindcss CSS framework.

## Installation and running

[GitHub Pages site](https://nojuskondrotas.github.io/a-eq-b/)

Install the dependencies:<br>
`bun i`

Build and run:<br>
`bun run dev`
